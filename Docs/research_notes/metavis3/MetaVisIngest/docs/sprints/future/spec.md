# MetaVisIngest - Specification

## Goals
1.  Robustly import media from various sources.
2.  Generate optimized proxies for editing.

## Requirements
- **Format Support**: Must support ProRes, H.264, HEVC, and Image Sequences.
- **Metadata**: Must extract full technical metadata (Color Space, Gamma, Frame Rate).
- **Proxies**: Must generate lightweight proxies for the Timeline.
